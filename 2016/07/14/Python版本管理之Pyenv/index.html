Python版本管理之Pyenv

发表于 2016-07-14   |  

由于某些原因，Python3并不兼容Python2，而在学习Python的过程中，经常需要使用不同的版本，因此，能够灵活安装和切换Python版本的pyenv就应运而生。

项目地址

github地址：pyenv

安装方法

在安装之前需要安装些依赖：

$ sudo apt-get update
$ sudo apt-get install make build-essential libssl-dev zlib1g-dev
$ sudo apt-get install libbz2-dev libreadline-dev libsqlite3-dev wget curl
$ sudo apt-get install llvm libncurses5-dev libncursesw5-dev

github地址：pyenv-installer
推荐方法：
安装:

$ curl -L https://raw.githubusercontent.com/yyuu/pyenv-installer/master/bin/pyenv-installer | bash

更新:

$ pyenv update

卸载：

$ rm -fr ~/.pyenv

由于作者并不推荐PyPi方式，故这里不介绍它，若想用该方式可以去pyenv-installer的github网站上查看。

常用指令

要学会使用终端的help命令
在终端中输入pyenv -h，结果如图所示：

由图可知，安装一个Python版本到使用的步骤如下：

  1. 首先应当使用pyenv install安装Python版本，可问题是，并不知道如何指定哪个版本，这时，使用Tab键可以给出提示，如图所示（内容太多，只截取部分）：

    然后在后面写上想要安装的版本即可。
  2. 安装之后，应当使用pyenv rehash来重新索引。
  3. 使用pyenv versions查看系统安装python版本。
  4. 启用安装的版本，使用pyenv local xxx或pyenv global xxx，其中xxx对应版本名，可使用Tab键给出提示，local和global的区别在pyenv -h所示的截图中有提到。

启用了安装的版本后，在终端执行python，pip等等命令时，其python版本自然就是所启用的版本了，通过次方法可以安装不同的版本，在各个版本中安装所需的库，然后随意切换而不会互相干扰。
